CogniBrew Airflow Orchestrator has been installed.

This chart uses the official Apache Airflow Helm chart as a dependency.

The following environment variables are configured for DAGs:
  • VECTOR_OP_URL  → Vector Operation service (cognibrew namespace)
  • EDGE_SYNC_URL  → Edge Sync service (cognibrew namespace)
  • S3_ENDPOINT_URL → RustFS (cognibrew-infra namespace)

To access the Airflow UI:
  kubectl port-forward svc/{{ .Release.Name }}-airflow-webserver 8080:8080 -n {{ .Release.Namespace }}
  Then visit: http://localhost:8080
  Login: airflow / airflow

To verify:
  kubectl get pods -n {{ .Release.Namespace }}
